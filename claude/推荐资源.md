## 推荐的MCP服务器

### deepwiki

[deepwiki](https://deepwiki.com/) 通过AI技术为GitHub代码仓库生成交互式文档和知识库，免费，非常推荐。

添加方式：

- 命令行：

  ```sh
  claude mcp add -s user -t http deepwiki https://mcp.deepwiki.com/mcp
  ```

- 完整配置文件：

  ```json
  {
    "mcpServers": {
      "deepwiki": {
        "type": "http",
        "url": "https://mcp.deepwiki.com/mcp"
      }
    }
  }
  ```

- cc-switch快速添加：

  ```json
  {
    "deepwiki": {
      "type": "http",
      "url": "https://mcp.deepwiki.com/mcp"
    }
  }
  ```
