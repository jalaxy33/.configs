# 阿里百炼设置

接入文档：<https://bailian.console.aliyun.com/cn-beijing/?tab=doc#/doc/?type=model&url=3023078>

推荐订阅coding plan，支持以下模型：`qwen3.5-plus`（支持图片理解）、`kimi-k2.5`（支持图片理解）、`glm-5`、`MiniMax-M2.5`、`deepseek-v3.2`

## 模型配置

Claude Code 环境变量 `~/.claude/settings.json`：

**注：请将 `"ANTHROPIC_AUTH_TOKEN"` 字段更换为自己的 API key**。

> 下面的 `ANTHROPIC_BASE_URL` 是 coding plan 的地址，如果没有订阅，换成 `https://dashscope.aliyuncs.com/apps/anthropic`

```json
{
  "env": {
    "ANTHROPIC_BASE_URL": "https://coding.dashscope.aliyuncs.com/apps/anthropic",
    "ANTHROPIC_AUTH_TOKEN": "<YOUR_API_KEY>",
    "API_TIMEOUT_MS": "3000000",
    "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": 1,
    "ANTHROPIC_MODEL": "qwen3.5-plus",
    "ANTHROPIC_SMALL_FAST_MODEL": "qwen3.5-plus",
    "ANTHROPIC_DEFAULT_SONNET_MODEL": "qwen3.5-plus",
    "ANTHROPIC_DEFAULT_OPUS_MODEL": "qwen3.5-plus",
    "ANTHROPIC_DEFAULT_HAIKU_MODEL": "qwen3.5-plus"
  }
}
```

## vscode 设置

  ```json

    "claude-code.selectedModel": "qwen3.5-plus"
  ```

## 可选：配置MCP服务器

**前提条件**：

1. 获取一个非coding plan的[百炼API Key](https://bailian.console.aliyun.com/cn-beijing/?tab=globalset#/efm/api_key)。
2. 前往百炼[MCP广场](https://bailian.console.aliyun.com/cn-beijing/?tab=app#/mcp-market)，搜索相应的MCP服务器（如**联网搜索**）并选择开通。

相关文档：

- [添加联网搜索MCP](https://bailian.console.aliyun.com/cn-beijing/?tab=doc#/doc/?type=model&url=3023217)
- [添加视觉理解能力](https://bailian.console.aliyun.com/cn-beijing/?tab=doc#/doc/?type=model&url=3023166)

### 方式一：cc-switch快速配置

点击 cc-switch 右上角添加MCP服务器。

> 注意将 `<your_api_key>` 换成非coding plan的[百炼API key](https://bailian.console.aliyun.com/cn-beijing/?tab=globalset#/efm/api_key)

- 百炼联网搜索：[开通链接](https://bailian.console.aliyun.com/cn-beijing/?tab=app#/mcp-market/detail/WebSearch)
  
  ```json
  {
    "AliWebSearch": {
      "type": "http",
      "url": "https://dashscope.aliyuncs.com/api/v1/mcps/WebSearch/mcp",
      "headers": {
        "Authorization": "Bearer <your_api_key>"
      }
    }
  }
  ```

- 百炼网页解析：[开通链接](https://bailian.console.aliyun.com/cn-beijing/?tab=app#/mcp-market/detail/WebParser)

  ```json
  {
    "AliWebParser": {
      "type": "sse",
      "url": "https://dashscope.aliyuncs.com/api/v1/mcps/WebParser/sse",
      "headers": {
        "Authorization": "Bearer <your_api_key>"
      }
    }
  }
  ```

- 百炼代码解释器：[开通链接](https://bailian.console.aliyun.com/cn-beijing/?tab=app#/mcp-market/detail/code_interpreter_mcp)

  ```json
  {
    "AliCodeInterpreter": {
      "type": "http",
      "url": "https://dashscope.aliyuncs.com/api/v1/mcps/code_interpreter_mcp/mcp",
      "headers": {
        "Authorization": "Bearer <your_api_key>"
      }
    }
  }
  ```

- 百炼TimeZone：[开通链接](https://bailian.console.aliyun.com/cn-beijing/?tab=app#/mcp-market/detail/TimeZone)

  ```json
  {
    "AliTimeZone": {
      "type": "sse",
      "url": "https://dashscope.aliyuncs.com/api/v1/mcps/TimeZone/sse",
      "headers": {
        "Authorization": "Bearer <your_api_key>"
      }
    }
  }
  ```

- 高德地图Amap：[开通链接](https://bailian.console.aliyun.com/cn-beijing/?tab=app#/mcp-market/detail/amap-maps)

  ```json
  {

      "type": "sse",

      "headers": {
        "Authorization": "Bearer <your_api_key>"

    }
  }
  ```

```sh
vim ~/.claude.json
```
